# 공식 근거 확인 보고서

## FACT

본 프로젝트는 CDC NHANES 2009-2010 공개 데이터 기반의 교육용 분석입니다.

사용 예정 파일:

| 파일 | 역할 |
|---|---|
| `ARQ_F.XPT` | 허리 통증 관련 자가보고 라벨 생성 |
| `PAQ_F.XPT` | 신체활동, 좌식시간 변수 |
| `BMX_F.XPT` | BMI, 허리둘레 등 신체계측 변수 |
| `SLQ_F.XPT` | 수면시간 변수 |
| `SMQ_F.XPT` | 흡연 변수 |
| `DEMO_F.XPT` | 나이, 성별, 교육, 소득 등 보정 변수 |

## INFERENCE

NHANES는 동일 개인 식별자인 `SEQN`을 기준으로 여러 설문/검진 파일을 병합할 수 있으므로, 생활습관 변수와 허리 통증 관련 문항을 개인 단위로 연결할 수 있습니다.

## ASSUMPTION

- 팀 프로젝트 시간 제약상 1차 구현은 NHANES 2009-2010만 사용합니다.
- 국민건강영양조사 기반 한국화는 후속 확장 과제로 둡니다.

## RECOMMENDATION
