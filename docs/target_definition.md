# 타겟 변수 정의 문서

## 목적

이 문서는 NHANES 2009-2010 데이터를 사용한 허리 통증 관련 신호 분석 프로젝트의 타겟 변수 정의를 고정하기 위한 문서입니다.

## FACT

- 병합 기준 변수는 `SEQN`입니다.
- 기본 타겟 후보는 Arthritis Questionnaire 파일(`ARQ_F.XPT`)의 허리 통증 관련 문항입니다.
- 프로젝트의 기본 타겟은 `ARQ020D == 4`를 양성 라벨로 정의합니다.
- `ARQ024D == 1`은 3개월 이상 지속 기준의 보조/민감도 분석 타겟으로만 사용합니다.

## RECOMMENDATION

기본 분석에서는 다음 라벨을 사용합니다.

```text
backpain_6w = 1 if ARQ020D == 4
backpain_6w = 0 if ARQ020D 응답 구조상 다른 부위 또는 비해당으로 해석 가능한 경우
결측/거절/모름/skip 구조는 코드북 기준으로 별도 검토 후 제외
```

## 주의 표현

사용 가능:

- 자가보고 허리 통증 라벨
- 6주 이상 지속된 low back pain 관련 문항 기반 라벨
- 허리 통증 관련 신호 분류

사용 금지:

- 의학적 진단
- 허리 통증 원인 판정
- 생활습관이 허리 통증을 유발한다
- 개인별 치료 또는 운동 처방

## 확인 필요

- `ARQ010`과 `ARQ020D`의 skip 구조를 실제 데이터와 코드북으로 재확인해야 합니다.
- 최종 표본 수와 양성 비율은 전처리 실행 후 확정해야 합니다.
