# LIGHT ONE BODY CARE SYSTEM

> 운동센터와 트레이너를 위한 비의료 웰니스 코칭 SaaS MVP

LIGHT ONE BODY CARE SYSTEM은 운동센터·트레이너가 회원의 자세, 통증 호소, 운동 수행 상태를 더 일관되게 기록하고 설명할 수 있도록 돕는 비의료 웰니스 코칭 시스템입니다. 이 저장소는 모두의창업 2차 검토를 위해 사업 방향, 데모 링크, 기술 구성, 검증 계획, 보완 이력을 한 곳에 정리하는 메인 창업 저장소로 재정렬됩니다.

본 프로젝트는 의료 진단, 치료, 처방, 임상 의사결정 지원을 목적으로 하지 않습니다. 결과물은 트레이너의 상담과 코칭을 보조하는 리포트 초안이며, 최종 판단과 설명은 사람 전문가가 검토하는 구조를 전제로 합니다.

## 1. 한 문장 정의

운동센터와 트레이너가 회원 상담 전후에 자세·통증 호소·운동 수행 기록을 정리하고, AI가 생성한 코칭 리포트 초안을 트레이너가 검토해 제공하는 비의료 웰니스 코칭 SaaS입니다.

## 3. 고객 문제

운동센터와 트레이너는 회원 상담에서 다음 문제를 반복적으로 겪습니다.

- 회원 상태 기록이 트레이너마다 다르고 누적 관리가 어렵습니다.
- 자세, 통증 호소, 운동 수행 품질을 설명할 때 객관적 근거를 보여주기 어렵습니다.
- 상담 리포트 작성 시간이 길어져 코칭 품질과 운영 효율이 떨어집니다.
- AI 도구를 쓰더라도 의료적 과장 표현이나 잘못된 권고가 포함될 위험이 있습니다.

LIGHT ONE은 이 문제를 “트레이너가 검토하는 AI 리포트 초안” 구조로 해결합니다.

## 4. 첫 고객 가설

1차 타깃은 1~3인 규모의 프리미엄 PT 스튜디오, 체형 교정·재활 운동 기반 운동센터, 회원 상담 리포트를 차별화 요소로 쓰고 싶은 트레이너입니다.

초기 판매 단위는 병원이나 개인 환자가 아니라 운동센터 또는 트레이너 계정입니다. 따라서 제품 표현은 의료 AI가 아니라 비의료 웰니스 코칭 보조 도구로 제한합니다.

## 5. 사용 흐름

```text
회원 기본 상담
→ 자세/통증 호소/운동 수행 정보 입력
→ BodyScan AI가 관찰 신호와 리포트 초안 생성
→ CADSS가 위험 표현과 검토 필요 항목을 분류
→ 트레이너가 최종 확인 및 수정
→ 회원용 웰니스 코칭 리포트 제공
```

## 6. 제품 구성

| 구성요소 | 역할 | 현재 상태 |
|---|---|---|
| LIGHT ONE 메인 페이지 | 사업 방향, 고객 문제, 서비스 흐름 제시 | Netlify 공개 페이지 운영 중 |
| BodyScan AI | 자세·통증 호소·운동 품질 분석 엔진 | 기술 데모 페이지 운영 중 |
| CADSS | AI 출력 위험도 관리, 사람 검토 라우팅 | 보완 페이지 운영 중 |
| GitHub 문서 | MVP, 검증 계획, 보완 이력 정리 | 본 저장소에서 재정렬 중 |
| 과거 허리통증 분석 | 공개 데이터 기반 교육용 실험 | 대표 주제가 아니라 참고 프로젝트로 격하 |

## 7. 수익모델 가설

- B2B SaaS 월 구독: 트레이너 또는 센터 단위 계정 과금
- 리포트 사용량 기반 과금: 월 기본 리포트 수 초과 시 추가 과금
- 프리미엄 온보딩: 센터용 초기 세팅, 리포트 템플릿 커스터마이징

정확한 가격은 파일럿 인터뷰와 지불의사 검증 전까지 확정하지 않습니다.

## 8. MVP 검증 계획

### 1단계: 문제 검증

- 트레이너 5명 이상 인터뷰
- 현재 상담 기록 방식, 리포트 작성 시간, 회원 설득 어려움 확인
- “AI 리포트 초안 + 트레이너 검토” 방식에 대한 수용성 확인

### 2단계: 사용성 검증

- Netlify 데모를 기준으로 상담 시나리오 3개 테스트
- 리포트 초안이 실제 상담 설명에 도움이 되는지 확인
- 위험 표현, 과장 표현, 불편한 표현을 수집해 CADSS 기준 개선

### 3단계: 지불의사 검증

- 무료 체험이 아니라 월 구독 또는 리포트 단가 기준으로 질문
- 센터장/트레이너가 비용을 지불할 명확한 이유 확인
- 결제 전환 전 최소 1개 센터 파일럿 후보 확보

## 9. 기술 구조

```text
사용자 입력
→ BodyScan AI 분석 엔진
→ 리포트 초안 생성
→ CADSS 위험도 라우팅
→ 트레이너 검토
→ 회원용 리포트 출력
```

BodyScan AI는 자세, 통증 호소, 운동 수행 정보를 바탕으로 관찰 가능한 웰니스 신호를 정리합니다. CADSS는 AI 출력에서 진단, 치료, 처방, 정확도 보장처럼 위험한 표현이 들어가는지 확인하고 사람 검토가 필요한 항목을 표시합니다.

## 10. 표현 원칙

사용 가능한 표현:

- 비의료 웰니스 코칭 SaaS
- 운동 코칭 보조
- 트레이너 검토 기반 리포트 초안
- 자세·통증 호소·운동 품질 관찰 신호
- 위험 신호 선별
- 파일럿 검증 예정
- 사람 전문가 최종 검토

피해야 할 표현:

- 진단
- 치료
- 처방
- 의료 AI
- 정확도 보장
- 통증 원인 판정
- 질병 예측
- 개인별 의료 판단

## 11. 과거 허리통증 프로젝트의 위치

허리통증 분석 프로젝트는 LIGHT ONE의 대표 주제가 아닙니다. 이 자료는 공개 데이터 기반 교육용 실험이자, 향후 BodyScan AI의 통증 호소 데이터 해석에서 참고할 수 있는 보조 자료로만 둡니다.

심사자에게 보여줄 때는 “메인 창업 아이템”이 아니라 “과거 데이터 분석 실험과 안전한 표현 원칙을 익힌 근거”로 설명합니다.
